## Installation

Clone repo

```sh
git clone https://github.com/isdi-coders-2023/Abel-Naharro-Final-Project-back-202309-mad
cd Abel-Naharro-Final-Project-back-202309-mad
```

Install dependencies

```sh
npm i
```

Compile TypeScript

```sh
npm run build
```

Run project

```sh
npm run start:dev
```

Run tests

```sh
npm run test:pro
```
